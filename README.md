## Usage

1. ```git clone git@github.com:newObjectccc/vtabs.git```
2. ```pnpm install```
3. ```pnpm start```

When the chrome launched, type "chrome://extensions" into the chrome address bar, then tap the Load unpacked button, and select the dist folder.
